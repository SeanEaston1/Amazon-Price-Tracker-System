<h1 align="center">Amazon Price Tracker</h1>

<p align="center">A system for tracking the prices of products on Amazon and receiving notifications when the price changes.</p>

## Features

- Real-time price updates
- User-friendly interface
- Efficient and reliable
- Option to set price alerts
- Easy to use and setup

## Requirements

- Web scraping technology, such as Beautiful Soup or Scrapy, for extracting product information and prices from Amazon's website.
- A programming language, such as Python, for building the system.
- A database, such as MySQL, for storing product information and price history.
- A web framework, such as Flask or Django, for building the user interface and server-side logic.
- Email and notification libraries, such as SMTP or Twilio, for sending alerts to users when the price of a product changes.

## Getting Started

- Clone the repository: git clone https://github.com/seaneaston1/amazon-price-tracker-system.git
- Install the required technologies/libraries: pip install -r requirements.txt
- Set up your Amazon API credentials
- Run the program: python main.py
